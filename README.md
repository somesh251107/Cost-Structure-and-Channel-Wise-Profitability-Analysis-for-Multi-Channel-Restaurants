# Cost-Structure-and-Channel-Wise-Profitability-Analysis-for-Multi-Channel-Restaurants
A comprehensive Financial Analytics and Business Intelligence project that analyzes cost structures and channel-wise profitability for multi-channel restaurants. The project evaluates In-Store, Uber Eats, DoorDash, and Self-Delivery operations using exploratory data analysis (EDA), financial KPIs, cost decomposition, profit margin analysis.
Project Overview

This project analyzes the financial performance of multi-channel restaurants by comparing profitability across In-Store, Uber Eats, DoorDash, and Self-Delivery channels. The analysis focuses on identifying the most profitable channel while understanding the impact of operating costs, commissions, and delivery expenses on business performance.

The project combines Financial Analytics, Business Intelligence, Exploratory Data Analysis (EDA), Machine Learning, and an interactive Streamlit dashboard to provide actionable insights for restaurant management.

🎯 Problem Statement

Restaurants often generate high sales across multiple channels but fail to identify which channels truly maximize profit.

Third-party delivery platforms increase order volume while charging significant commissions, whereas self-delivery introduces logistics costs. This project evaluates these trade-offs to support better business decisions.
